# 🚚 MiniCoreLogistica

> **Smart Logistics Engine:** Solución ágil y desacoplada para la orquestación, tracking y gestión de distribución de carga de última milla. Desarrollado bajo arquitectura multicapa en el ecosistema .NET.

---

## ⚡ Capacidades del Núcleo

*   **⚡ Despacho Automatizado:** Motor modular para el aprovisionamiento y registro de Envíos en tiempo real.
*   **👥 CRM Nativo Flatafórmico:** Gestión centralizada y persistente de Clientes con tipado seguro.
*   **🚴 Fleet Management:** Control de estados, disponibilidad y asignación inteligente de Repartidores.
*   **💻 Client-Side UI:** Interfaz web enriquecida, minimalista y reactiva mediante Vanilla JavaScript.

---

## 🛠️ Stack Arquitectónico

| Componente | Tecnología | Rol en el Sistema |
| :--- | :--- | :--- |
| **Backend** | `.NET 6 (C#)` | API Restful con Inyección de Dependencias nativa. |
| **Persistencia** | `EF Core` | Capa ORM para el mapeo relacional automatizado (Code-First). |
| **Frontend** | `HTML5 / CSS3 / ES6+` | Single Page Application (SPA) desacoplada y asíncrona (`Fetch API`). |

# 1. Clonar el ecosistema
