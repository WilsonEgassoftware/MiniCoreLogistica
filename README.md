# 🚚 MiniCoreLogistica

Sistema básico de logística desarrollado en ASP.NET Core + Frontend HTML.

## 📌 Funcionalidades
- Registro de clientes
- Registro de envíos
- Gestión de repartidores
- Interfaz web simple

## 🛠️ Tecnologías
- C# .NET 6
- Entity Framework Core
- HTML + CSS + JavaScript

## 🚀 Ejecución
1. Clonar repositorio
2. Instalar paquetes NuGet
3. Ejecutar proyecto
